# cyber-project-14-ioc-detection-system
IOC-based threat detection system using Python (SOC simulation)


## Context

Security teams use Indicators of Compromise (IOCs) to detect known threats.

This project simulates detecting malicious activity using IOC matching.

---

## Objective

* Load IOC data (IPs, files)
* Scan logs for matches
* Generate alerts

---

## ⚙️ Features

* IOC matching
* Modular architecture
* Threat detection
